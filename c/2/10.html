테이블 만들기

이번에는 HTML로 테이블을 만드는 방법을 배워보겠습니다. 테이블을 사용하면 데이터를 구조적으로 정리하여 표현할 수 있습니다. 테이블을 만드는 기본 코드는 아래와 같습니다:

Header 1 Header 2 Header 3
Data spanning two columns Data 3
Data spanning two rows Data 2 Data 3
Data 2.1 Data 3.1

위의 코드는 다양한 속성을 사용한 테이블의 예시입니다. 이제 테이블 요소와 속성에 대해 자세히 알아보겠습니다.

요소 설명 필수 여부
<table> 테이블의 전체 구조를 정의하는 요소입니다. border 속성은 테이블의 테두리를 설정하며, align 속성은 테이블의 정렬을 설정합니다. 예: <table border="1" align="center"> O
<tr> 테이블의 한 행(row)을 정의하는 요소입니다. O
<th> 테이블의 헤더 셀을 정의하는 요소로, 보통 굵게 표시되고 가운데 정렬됩니다. X
<td> 테이블의 데이터 셀을 정의하는 요소입니다. O
align 테이블 또는 셀의 내용을 좌, 우, 중앙에 정렬할 수 있습니다. 예: <td align="center"> X
border 테이블의 셀과 셀 사이에 테두리를 설정합니다. 숫자가 클수록 테두리가 두꺼워집니다. 예: border="1" X
colspan 셀 하나가 여러 열(column)을 차지할 수 있도록 설정합니다. 예: <td colspan="2">는 셀이 두 개의 열에 걸쳐 있음을 나타냅니다. X
rowspan 셀 하나가 여러 행(row)을 차지할 수 있도록 설정합니다. 예: <td rowspan="2">는 셀이 두 개의 행에 걸쳐 있음을 나타냅니다. X

테이블에서는 colspan과 rowspan 속성을 사용하여 셀이 여러 열이나 행에 걸쳐 나타나도록 할 수 있습니다. 이 속성들을 사용하면 복잡한 테이블 레이아웃을 쉽게 만들 수 있습니다.

이제 직접 예시 코드를 사용하여 다양한 형태의 테이블을 만들어 보세요!
