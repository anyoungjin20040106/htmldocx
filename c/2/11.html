위의 코드는 다양한 속성들의 입력 요소들과 버튼입니다

form 태그 분석

요소 설명
action 제출 후 갈 위치 입니다
method 제출 방식입니다(Get, Post)

제출방식

이름 설명
Get URL에 변수명과 값을 넣어서 보내는 방식입니다
Get URL에 변수명과 값을 넣지않고 보내는 방식입니다

input 태그 분석

요소 설명 필수 여부
type 입력 종류 입니다. 글자를 입력받거나 체크박스등 다양한 종류를 지정할 수 있습니다(기본값 : text) X
name 해당 요소의 이름입니다 O
value 해당 요소의 값 입니다 X

input의 type종류

종류 설명 생김새
text 글자를 입력받기 위한 입력요소 입니다
password 비밀번호를 입력받기 위한 입력요소입니다
number 숫자를 입력받기 위한 입력요소입니다
checkbox 체크박스로 name이 같을 때 여러개의 항목을 선택할수 있습니다
radio 체크박스랑 비슷하지만 한개의 이름에 한개의 값만 저장할 수 있습니다
