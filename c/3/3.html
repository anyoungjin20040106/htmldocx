입력 요소의 값을 가져와보자

이번에는 입력요소에 값을 가져와서 값을 구하는 사이트를 만들어 보겠습니다


반지름 :

코드 분석

요소 설명
id - HTML 요소의 고유 식별자입니다.
- 자바스크립트에서 해당 요소를 찾거나 조작할 때 사용합니다.
document.getElementById('id').value 특정 id를 가진 요소를 찾아 그 요소의 값을 가져옵니다.
document.getElementById('id').innerHTML 특정 id를 가진 요소의 내부 HTML 내용을 가져오거나 설정합니다.
<div> 특정 id가 담긴 태그입니다
