SQL이란?

SQL은 Struct Query Language의 줄임말로 한국어로 말하자면 구조적인 데이터에게 명령(삽입, 수정, 삭제, 조회)을 내리는 언어입니다.
SQL의 큰 특징은 DB(DataBase)와 테이블로 이루어져 있다는 특징입니다 테이블은 정보를 저장하는 공간이고
DB는 테이블을 모아놓은 공간으로 다양한 테이블을 연결할수도 있습니다.
SQL의 종류는 아래와 같습니다.

SQL명설명
MySQL MySQL은 오픈 소스 관계형 데이터베이스 관리 시스템으로, 속도와 신뢰성이 뛰어나고 많은 웹 애플리케이션에서 사용됩니다.
Oracle Oracle Database는 오라클 사에서 개발한 상용 관계형 데이터베이스 관리 시스템으로, 대규모 엔터프라이즈 환경에서 많이 사용됩니다.
MSSQL Microsoft SQL Server는 마이크로소프트에서 개발한 상용 관계형 데이터베이스 관리 시스템으로, 윈도우 환경에서 주로 사용됩니다.
PostgreSQL PostgreSQL은 오픈 소스 객체-관계형 데이터베이스 시스템으로, 복잡한 쿼리와 대용량 데이터를 처리하는 데 강력한 기능을 제공합니다.
SQLite SQLite는 경량의 관계형 데이터베이스 관리 시스템으로, 서버가 필요 없는 임베디드 데이터베이스로 많이 사용됩니다.
