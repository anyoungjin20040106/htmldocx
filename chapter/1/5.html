서버프로그래밍이란?

웹개발은 html과 css, javascript로 이루어져있다고 말하지만 그것은 프론트엔드(웹의 디자인이나 요소)한정으로 말하는것이고 백엔드같은 경우에는 서버와 SQL을 알아야합니다.
서버는 DB와 웹을 연동해주는 역할로 사용하는 언어는 아래와 같습니다

언어설명
golang Golang, 또는 Go는 구글에서 개발한 오픈 소스 프로그래밍 언어로, 성능과 효율성을 강조하며, 서버 사이드 개발에 많이 사용됩니다.
php PHP는 서버 사이드 스크립트 언어로, 웹 개발을 위해 설계되었으며, 동적 웹 페이지를 생성하는 데 많이 사용됩니다.
python Python은 간결하고 읽기 쉬운 문법을 가진 고급 프로그래밍 언어로, 대표적으로 Fastapi가 있습니다.
java Java는 객체 지향 프로그래밍 언어로, 대표적으로 jsp가 있습니다
