HTML 태그 표

태그 설명
<a> 하이퍼링크를 정의합니다. href 속성을 사용하여 링크할 URL을 지정합니다. target="_blank"를 추가하면 새탭에서 해당 URL이 열립니다
<img> 이미지를 삽입합니다. src 속성을 사용하여 이미지의 경로를 지정합니다.
<center> 모든 요소들을 가운데 정렬 합니다
<h1> 글자를 제일 굵게 합니다
<h2> 글자를 2번쨰로 굵게 합니다
<h3> 글자를 3번쨰로 굵게 합니다
<h4> 글자를 4번쨰로 굵게 합니다
<h5> 글자를 5번쨰로 굵게 합니다
<h6> 글자를 6번쨰로 굵게 합니다
